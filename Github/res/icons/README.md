# Icons for Gentoo-Update

This directory contains SVG icons for Gentoo-Update in various styles.

## Icon Variants

- `gentoo-update.svg` - Standard dark icon
- `gentoo-update-light.svg` - Light variant for dark backgrounds  
- `gentoo-update-blue.svg` - Blue variant
- `gentoo-update_updates-available.svg` - Indicates updates available
- `gentoo-update_updates-available-light.svg` - Light variant
- `gentoo-update_updates-available-blue.svg` - Blue variant

These icons are used by:
- System tray applet
- Desktop environment integration
- Application menus
