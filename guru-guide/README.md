This guide covers the complete workflow for submitting metetch (v0.5.3) to Gentoo GURU:

**Estimated Time**: 2-3 hours for initial setup, 30 minutes for submission
**Difficulty**: Intermediate (requires Git, OpenPGP knowledge)
**Requirements**: SSH access, GPG key, Gentoo system
